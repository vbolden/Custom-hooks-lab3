## Scenario

You’ve been tasked with developing two common custom Hooks: usePagination for managing client-side pagination logic, and useDebounce for debouncing rapidly changing values, often used with search inputs. This lab will challenge you to think about hook design, state management within hooks, handling dependencies, and ensuring your hooks are generic enough for reuse.

### Built with

- CSS custom properties
- [React](https://reactjs.org/) - TS library
